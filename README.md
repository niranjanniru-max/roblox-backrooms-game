# Backrooms Escape Game (Roblox)

## Project Overview
Backrooms Escape is a survival horror game developed on Roblox where the player must escape from a monster inside a closed environment. The project focuses on game logic, scripting, and performance handling.

## Features
- Player movement and controls
- Monster enemy behavior and spawning
- Health and damage system
- Escape and lose conditions
- Increasing difficulty during gameplay

## Technologies Used
- Roblox Studio
- Lua scripting

## My Role
- Designed and implemented core gameplay mechanics
- Wrote Lua scripts for player and enemy behavior
- Implemented game rules such as win and lose conditions
- Debugged logic issues and improved performance

## Challenges & Solutions
- Faced logic bugs, performance drops, and scripting issues
- Solved problems by testing scripts, optimizing logic, and refining enemy behavior

## Current Status
- Playable game
- Actively improving features and performance

## What I Learned
- Game logic and scripting using Lua
- Debugging and performance optimization
- Building a complete playable game from scratch
